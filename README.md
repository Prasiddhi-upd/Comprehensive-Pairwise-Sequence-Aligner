# Comprehensive-Pairwise-Sequence-Aligner

A Python toolkit for pairwise sequence alignment implementing global, local, semi-global, and affine gap algorithms. Supports substitution matrices, PAM generation, and nucleotide-to-protein translation.

## Features

- Global (Needleman-Wunsch), Local (Smith-Waterman), Semi-Global, and Affine Global alignment.
- Read sequences from files or direct input.
- Optional PAM-n substitution matrix generation.
- Nucleotide-to-protein translation.

- Python 3.7+
- NumPy
